# Mis Tesoros

- ASP.Net Core
- Blazor Server
- Entity Framework
- Identity
- Google Maps

## Arrancar Proyecto

1. Descargar e instalar dotnet core <https://dotnet.microsoft.com/download>
2. ```bash dotnet restore ```
3. ```bash dotnet ef database update ```
4. ir a Pages/_Host.cshtml y agregar el api key de google maps
5. ```bash dotnet run ```
