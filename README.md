# Mis Tesoros

- ASP.Net Core
- Blazor Server
- Entity Framework
- Identity
- Google Maps

## Arrancar Proyecto

- Descargar e instalar dotnet core <https://dotnet.microsoft.com/download>
- Desde la terminal:
```bash
git clone uri-repo
cd proyecto
```
- Ir a Pages/_Host.cshtml y agregar el api key de google maps
- Desde la terminal:
```bash 
dotnet restore 
dotnet ef database update
dotnet run
```
